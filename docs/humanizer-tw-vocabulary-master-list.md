### 前提修正

Gemini 花了大量篇幅分析「可能誤判句」（word boundary 問題），模擬正則表達式機械替換的風險。但 humanizer-tw 是 **LLM 讀取字典後語境理解式改寫**，不是 string replace。LLM 天然理解詞界，不會把「袋鼠標本」切成「袋滑鼠本」。因此「替換風險」欄位在我們的使用情境中可以忽略，回歸看**頻率 × 辨識度**。

## 六、中台用語差異的系統性規律

理解差異的底層規律，有助於發現新的遺漏詞。

### 1. 音譯 vs 意譯

中國傾向音譯，台灣傾向意譯：

| 中國（音譯） | 台灣（意譯） | 原文 |
|-------------|-------------|------|
| 博客 | 部落格 | blog |
| 黑客 | 駭客 | hacker |
| 芯片 | 晶片 | chip |

### 2. 簡化 vs 完整

中國傾向縮寫或簡化，台灣保留較完整的詞：

| 中國（簡化） | 台灣（完整） | 原文 |
|-------------|-------------|------|
| 信息 | 資訊 | information |
| 軟件 | 軟體 | software |
| 硬件 | 硬體 | hardware |

### 3. 不同的漢字選擇

同一概念選用不同漢字：

| 中國 | 台灣 | 對應概念 |
|------|------|---------|
| 數據 | 資料 | data |
| 視頻 | 影片 | video |
| 網絡 | 網路 | network |
| 內存 | 記憶體 | memory (RAM) |

### 4. 動詞偏好差異

| 中國偏好 | 台灣偏好 | 語境 |
|---------|---------|------|
| 運行 | 執行 | run a program |
| 調試 | 除錯 | debug |
| 激活 | 啟用 | activate |
| 配置 | 設定 | configure |
| 登錄 | 登入 | log in |

### 5. 口語/網路用語的完全分歧

| 中國 | 台灣 | 備註 |
|------|------|------|
| 小夥伴 | 同事、朋友 | 台灣不會這樣稱呼 |
| 給力 | 讚、很強 | 台灣偶見但不自然 |
| 靠譜 | 可靠、穩 | 台灣幾乎不用 |
| 點贊 | 按讚 | 功能相同，說法不同 |

### 6. 制度/生活用語差異

| 中國 | 台灣 | 備註 |
|------|------|------|
| 初中 | 國中 | 教育制度用語 |
| 高考 | 大學入學考試 | 但台灣「高考」指公務員考試，不可替換 |
| 出租車 | 計程車 | 交通用語 |
| 公交車 | 公車 | 交通用語 |
| 短信 | 簡訊 | 通訊用語 |
| 外賣 | 外送 | 生活服務用語 |
