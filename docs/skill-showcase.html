#2 什麼是 Agent Teams（定義段） 拆句 · 重組結構

不像 subagent 只回報結果——teammates 能互相傳訊、挑戰彼此結論、在對方發現的基礎上推進。這才是真正的突破。

humanizer-tw

跟 subagent 差很多。subagent 做完回報就結束了，teammates 會互相傳訊、挑戰對方結論、接著對方的發現繼續往下做。差別在這裡。

good-writing-zh 改了什麼：去弱詞——「這種協作能力才是真正的突破點」濃縮成「這才是真正的突破」，六個字少了但力度反而更強。
