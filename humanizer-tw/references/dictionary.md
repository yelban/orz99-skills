## 書面語 → 口語

### 書面代詞

| 避免 | 改用 |
|------|------|
| 其 | 它的、他的 |
| 該 | 這個、那個 |
| 此 | 這、這個 |
| 予以 | 給、進行 |
| 之 | 的（或省略） |

### 介詞結構

| 避免 | 改用 |
|------|------|
| 對於...而言 | 對...來說（或省略） |
| 就...來說 | 說到...（或省略） |
| 在...方面 | ...上（或省略） |
| 基於...的考量 | 考慮到... |
| 鑑於...的情況 | 因為... |

### 動詞結構

| 避免 | 改用 |
|------|------|
| 進行了...的工作 | 做了... |
| 開展了...活動 | 辦了... |
| 實現了...的目標 | 達成... |
| 取得了...的成果 | 有了... |
| 做出了...的貢獻 | 幫了... |

### 正式 → 口語

| 正式 | 口語 |
|------|------|
| 我認為 | 我覺得 |
| 需要考慮 | 得想想 |
| 進行討論 | 聊一下 |
| 提出建議 | 說個想法 |
| 給予回應 | 回一下 |

| 從不 | 很少、幾乎不 |
| 每個人 | 很多人、大部分人 |
| 沒有人 | 很少人 |
| 一定會 | 可能會、應該會 |
| 絕對不會 | 不太會 |

### AI 高頻強調詞

刪除或減少使用：

- 深深地、真正地、從根本上、本質上
- 簡單地、字面上、必然地、確確實實
- 實際上、事實上、嚴格來說

## 快速對照

| 類型 | 例子 | 處理 |
|------|------|------|
| 商業術語 | 「賦能」「痛點」「底層邏輯」 | 用簡單詞替換 |
| 書面語 | 「予以」「該」「鑑於」 | 用口語替換 |
| 中國用語 | 「視頻」「默認」「數據」「營銷」「概率」 | 換成台灣說法 |
| 絕對詞 | 「總是」「所有人」 | 軟化 |
