# 中文 AI 高頻短語

## 開場白套話

直接刪除這些，從內容開始。

### 時代開場
- 「隨著...的發展」「隨著...的興起」「隨著...的普及」
- 「在...的背景下」「在...的浪潮中」
- 「當今時代」「在這個...的時代」「當下」
- 「進入...時代」「踏入...的新紀元」

### 共識開場
- 「眾所周知」「不言而喻」「顯而易見」
- 「毋庸置疑」「不可否認」「毫無疑問」
- 「正如大家所知」「想必大家都知道」

### 宣告式開場
- 「值得一提的是」「值得注意的是」「值得關注的是」
- 「不得不說」「必須承認」「坦白說」
- 「有趣的是」「有意思的是」「令人驚訝的是」

---

## 連接詞濫用

AI 文章過度依賴這些詞來「串連」段落，刪除或減少使用。

### 遞進連接
- 「此外」「另外」「不僅如此」「除此之外」
- 「更重要的是」「更為關鍵的是」「更值得關注的是」
- 「與此同時」「同時」「另一方面」

### 轉折連接
- 「然而」「但是」「不過」「儘管如此」
- 「話雖如此」「話說回來」「反過來看」

### 列舉連接
- 「首先...其次...再次...最後...」
- 「第一...第二...第三...」
- 「一方面...另一方面...」

### 總結連接
- 「總的來說」「總而言之」「綜上所述」
- 「歸根結底」「說到底」「從本質上來說」

## 結尾套話

直接刪除或用具體結論替代。

### 展望類
- 「讓我們拭目以待」「未來可期」「前景可期」
- 「相信在...的努力下」「期待...的到來」
- 「讓我們共同期待」「讓我們一起見證」

### 口號類
- 「為...貢獻力量」「為...添磚加瓦」
- 「讓我們攜手共進」「讓我們並肩前行」
- 「開啟...的新篇章」「書寫...的新故事」

### 反思類
- 「這是一個值得思考的問題」「這值得我們深思」
- 「或許，答案就在...」「也許，這就是...的意義」
- 「這需要時間來驗證」「時間會給出答案」

### 許可類
- 「這沒關係」「這是正常的」「這很常見」
- 「不要擔心」「不必焦慮」

## 書面語過重

用口語替換這些過於正式的詞。

### 代詞
| 避免 | 改用 |
|------|------|
| 其 | 它的、他的 |
| 該 | 這個、那個 |
| 此 | 這、這個 |
| 予以 | 給、進行 |
| 之 | 的（或省略） |

### 介詞結構
| 避免 | 改用 |
|------|------|
| 對於...而言 | 對...來說（或省略） |
| 就...來說 | 說到...（或省略） |
| 在...方面 | ...上（或省略） |
| 基於...的考量 | 考慮到... |
| 鑑於...的情況 | 因為... |

### 動詞結構
| 避免 | 改用 |
|------|------|
| 進行了...的工作 | 做了... |
| 開展了...活動 | 辦了... |
| 實現了...的目標 | 達成... |
| 取得了...的成果 | 有了... |
| 做出了...的貢獻 | 幫了... |
