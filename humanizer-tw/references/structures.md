# 中文 AI 結構問題

## 翻譯腔結構

AI 模型訓練資料中有大量翻譯文本，導致中文輸出帶有翻譯腔。

### 「這是一個...的事情」結構

英文 "It is a... thing" 的直譯。

| 翻譯腔 | 自然中文 |
|--------|----------|
| 這是一個很重要的事情 | 這很重要 |
| 這是一個值得探討的問題 | 值得探討 |
| 這是一個令人興奮的消息 | 令人興奮 |
| 這是一個不容忽視的現象 | 不能忽視 |

### 過長的定語從句

英文 relative clause 的影響，在名詞前堆疊過多修飾語。

| 翻譯腔 | 自然中文 |
|--------|----------|
| 在這個充滿機遇與挑戰並存的時代裡的年輕人 | 這個時代的年輕人面對機遇和挑戰 |
| 一個能夠幫助用戶快速完成任務的工具 | 一個工具，能幫用戶快速完成任務 |
| 那些在過去幾年中取得顯著成就的公司 | 過去幾年表現好的公司 |

### 「的」字堆疊

| 翻譯腔 | 自然中文 |
|--------|----------|
| 我的朋友的公司的產品的設計 | 我朋友公司產品的設計 |
| 這個專案的核心的功能的實現 | 這個專案核心功能的實現 |
| 用戶的需求的滿足的程度 | 用戶需求的滿足程度 |

**原則：** 連續超過兩個「的」就要重寫。

### 被動語態過度使用

| 翻譯腔 | 自然中文 |
|--------|----------|
| 這個問題被認為是 | 大家認為這個問題是 |
| 這項政策被廣泛討論 | 大家都在討論這項政策 |
| 報告被提交給管理層 | 我們把報告交給管理層 |
| 這個方案被採納了 | 我們採納了這個方案 |

## 公式化段落結構

### 開頭-中間-結尾公式

AI 喜歡這種結構：
1. 開頭：總結性陳述
2. 中間：三點展開
3. 結尾：昇華金句

**問題範例：**
> 「人工智慧正在改變世界。首先，它提高了生產效率。其次，它改變了人們的生活方式。最後，它帶來了新的商業機會。毫無疑問，AI 將繼續深刻影響人類社會。」

**改寫：**
> 「AI 讓工廠產能翻倍，我媽現在用語音助手訂菜，創業公司靠它拿到融資。」

### 三段式論證

每個觀點都用「論點→論據→結論」結構，讀起來像考試作文。

**問題範例：**
> 「遠端工作有許多優點。研究表明，遠端工作可以提高生產力。因此，越來越多的公司開始採用遠端工作模式。」

**改寫：**
> 「在家工作，我早上省了一小時通勤，下午多寫了兩個功能。公司看到產出沒掉，也就不管了。」

### 否定式排比

「不是 X，而是 Y」「不僅 X，更是 Y」的機械堆疊。

| 問題模式 | 處理方式 |
|----------|----------|
| 「不是 A，而是 B」 | 直接說 B |
| 「不僅 A，更是 B」 | 說「A，也是 B」或只說 B |
| 「不單單是 A，還是 B，更是 C」 | 挑最重要的一個說 |

## 語氣問題

### 過度正式

AI 預設輸出正式書面語，缺乏口語感。

| 正式 | 口語 |
|------|------|
| 我認為 | 我覺得 |
| 需要考慮 | 得想想 |
| 進行討論 | 聊一下 |
| 提出建議 | 說個想法 |
| 給予回應 | 回一下 |

### 千篇一律的客套

| 客套 | 處理 |
|------|------|
| 「非常感謝您的閱讀」 | 刪除 |
| 「希望這篇文章對您有所幫助」 | 刪除 |
| 「如有不足之處，請多多指教」 | 刪除 |
| 「歡迎在評論區留言」 | 刪除或自然帶過 |

### 缺乏個人觀點

AI 傾向使用「有人認為」「專家指出」等規避主觀立場。

| 規避 | 直接 |
|------|------|
| 「有人認為這很重要」 | 「這很重要」 |
| 「專家指出這是趨勢」 | 「這是趨勢」 |
| 「研究表明效果顯著」 | 「效果顯著」或引用具體研究 |
| 「人們普遍認為」 | 「我覺得」或「大家覺得」 |

## 節奏問題

### 句子長度單一

AI 輸出的句子長度常常很均勻，缺乏變化。

**問題範例：**
> 「這個產品很好用。它的設計很精美。價格也很合理。用戶評價很高。」

**改寫：**
> 「產品好用，設計精美。價格合理，評價高——買就對了。」

### 段落節奏機械

每段 3-4 句，每句 15-20 字，讀起來像機器打字。

**修正原則：**
- 長短句交替
- 偶爾用一兩個字的段落
- 不要每段都一樣長

### 過渡詞依賴

AI 不信任讀者能跟上思路，每個轉折都要加連接詞。

**問題範例：**
> 「首先，我們需要理解問題。其次，我們要分析原因。然後，我們制定方案。最後，我們執行計畫。」

**改寫：**
> 「理解問題。分析原因。制定方案。執行。」

## 詞彙模式

### 絕對詞過度使用

| 避免 | 改用 |
|------|------|
| 總是 | 常常、通常 |
| 從不 | 很少、幾乎不 |
| 每個人 | 很多人、大部分人 |
| 沒有人 | 很少人 |
| 一定會 | 可能會、應該會 |
| 絕對不會 | 不太會 |

### AI 高頻強調詞

這些詞 AI 用得特別多，刪除或減少：

- 深深地、真正地、從根本上、本質上
- 簡單地、字面上、必然地、確確實實
- 實際上、事實上、嚴格來說

### 同義詞機械替換

AI 為了避免重複，會刻意替換同義詞，反而不自然。

**問題範例：**
> 「這位工程師→這名技術人員→該名從業者→這位專業人士」

**原則：** 重複沒關係。用同一個詞比硬換更自然。

## 快速檢查

| 問題 | 徵兆 | 修正 |
|------|------|------|
| 翻譯腔 | 「這是一個...的」 | 刪除框架，直接說 |
| 定語過長 | 名詞前超過 10 字修飾 | 拆成多句 |
| 「的」堆疊 | 連續 3 個以上「的」 | 重寫句子 |
| 被動語態 | 「被」字過多 | 改成主動句 |
| 公式化 | 首先...其次...最後 | 打亂順序或刪除標記 |
| 語氣問題 | 讀起來像公文 | 加入口語表達 |
| 節奏單一 | 每句差不多長 | 長短交替 |
